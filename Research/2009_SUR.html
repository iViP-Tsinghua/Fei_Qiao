　 Overview
　

This project will include some research works on heterogeneous multi-core architecture exploring for multimedia applications, especially the video processing applications, which would be cooperative with IBM¨s internal multi-core architecture projects. Moreover, with the prevailing of multimedia applications and the forthcoming multi-core technologies, such application-specific heterogeneous multi-core architecture would have the better performance while using less resource than the common architectures, which means less power consumptions. Besides, with the development of DSM (Deep Submicron Meter) technology, billions of transistors would be fabricated on a single chip, which makes the implementation of such architectures possible.
